# Streaming App

> This project is a mini web app, where you can signed in using google and create stream channels.

## Built With

- Javascript
- React
- Redux
- Semantic UI
- Node Media Server
- JSON Server

![streaming](https://user-images.githubusercontent.com/11781597/83786407-e1414600-a64f-11ea-844b-337b1a5e5c25.png)

## Installation

1. Clone the repository in your local folder.
```
git clone https://github.com/Juakata/Streaming-App-Client.git
```
2. Run
```
npm install
```
```
npm start
```
3. To have the project completely working also you need to have this other 2 repositories:

- [API](https://github.com/Juakata/Streaming-App-API).
- [Node Media Server](https://github.com/Juakata/Streaming-App-Server).
